03 / Math / Algorithm Used

Math / Algorithm Used

No numerical algorithms. The work is structural: mapping structured_content dicts to OOXML table rows using python-docx primitives. Key structural rules:

  • Header rows get shading D9EAF7 and repeat via w:tblHeader
  • First column of 2-column tables gets shading F2F6FA
  • All rows get w:cantSplit to prevent page breaks mid-row
  • Cell margins default to 90 twips (~1.6 mm) for compact VCS layout
  • Font sizes vary by table density: 7.2pt (dense GHG boundary) to 9.3pt (cover metadata)

06 / Limitations / Second-Best Alternative

Limitations / Second-Best Alternative

  • Template style drift: The Verra template lacks several built-in styles present in a blank Document (e.g. Light Grid Accent 1, List Bullet). We added safe fallback setters rather than injecting custom styles into the template.
  • Template licensing: The template is Verra-owned; we distribute it in-repo for fidelity but note it is the official VCS v4.4 template.
  • Alternative rejected (ALT-001): Importing the Codex monolithic script as a separate legacy exporter was rejected to avoid parallel maintenance paths.
