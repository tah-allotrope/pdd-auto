06 / Limitations / Second-Best Alternative

Limitations / Second-Best Alternative

  • Quantification TBD: All emission-reduction numbers are intentionally null because the validated calculation spreadsheet was not in the Codex intake. Phase 03 must populate these or mark them explicitly.
  • Stakeholder consultation missing: The DOCX states consultation was not performed; this is a real gap that would block VCS validation.
  • EIA not completed: Environmental impact assessment is flagged as not done — another validation blocker.
  • Second-best alternative: Instead of reverse-engineering from DOCX, we could have requested the original intake YAML from the Codex session. That would have been faster and more accurate, but the file was not available.
07 / Errors / Warnings / Flags

Errors / Warnings / Flags Encountered

  • YAML parse error (fixed): Nested quotes in eligibility checklist key broke PyYAML: "Applicability ... "ACM0022"": true. Fixed by switching outer quotes to single quotes.
  • Summary NoneType format error (fixed): summary() crashed when net_emissions_tco2e_per_year was None. Fixed by adding conditional formatting (TBD fallback).
  • No blockers: All 190 tests pass; schema changes are backward-compatible.
