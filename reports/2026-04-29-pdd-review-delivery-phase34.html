06 / Limitations / Second-Best Alternative

Limitations / Second-Best Alternative

  • The draft is now discoverable and uploadable, but the content still carries one blocking review issue and all sections remain in domain review state.
  • The uploader currently returns a Drive folder URL fallback rather than a direct file URL unless stdout includes the file URL explicitly.
  • Second-best alternative would have been to keep upload tied to data/runs/{run_id}.docx, but that would split the review surface from the shareable surface again.
