06 / Limitations / Second-Best Alternative

Limitations / Second-Best Alternative

Limitations

  • The benchmark still runs on `NoopProvider`, so every section remains placeholder text with `UNSUPPORTED` confidence.
  • The reference comparison measures structural grounding and provenance overlap, not domain correctness of final prose.
  • One Soc Son-like case is useful as workflow proof, but not enough to support broader WTE generalization.

Second-best alternative

  • A lighter version would only add a shell script around existing commands and skip benchmark scoring.
  • That would be faster, but it would miss the Phase 05 requirement for measurable comparison, decision support, and durable artifacts.
07 / Errors / Warnings / Flags

Errors / Warnings / Flags Encountered

  • Initial test failure: `ModuleNotFoundError: No module named 'pdd_agent.phase05'` until the new Phase 05 package was implemented.
  • Smoke-run failure: `ImportError` from `SectionOrchestrator.run_review()` trying to import `ReviewState` from the wrong module; fixed by removing the incorrect import.
  • Runner bootstrap failure: `scripts/run_demo.py` needed repo-root path setup so it could run directly without assuming editable-install import behavior.
  • DOCX export caveat: the smoke run logged that `python-docx` is not installed in the current environment, so benchmark artifacts were generated without DOCX output.
